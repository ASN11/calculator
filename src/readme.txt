SmartCalc v2.0

Программа имеет 3 режима: калькулятор, кредитный калькулятор, депозитный калькулятор

Калькулятор содержит следующие функции:
Скобки (a + b);
Сложение a + b
Вычитание a - b
Умножение a * b
Деление a / b
Возведение в степень a ^ b
Остаток от деления a mod b
Унарный плюс +a
Унарный минус -a
Вычисляет косинус cos(x)
Вычисляет синус sin(x)
Вычисляет тангенс tan(x)
Вычисляет арккосинус acos(x)
Вычисляет арксинус asin(x)
Вычисляет арктангенс atan(x)
Вычисляет квадратный корень sqrt(x)
Вычисляет натуральный логарифм ln(x)
Вычисляет десятичный логарифм log(x)

Кредитный калькулятор рассчитывает на основе полученных данных ежемесячный платеж, переплату по кредиту, общую выплату

Депозитный калькулятор рассчитывает на основе полученных данных начисленные проценты, сумму на вкладе к концу срока

Установка:
make install - установка программы
make uninstall - удаление программы
make clean - удаление временных файлов установки
make dvi - открытие документации
make dist - сборка всех установочных файлов в отдельную директорию
make tests - запуск unit тестов
make gcov_report - формирование отчета по итогам тестирования
